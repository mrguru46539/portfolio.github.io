# Personal Portfolio Website

A modern, responsive, and performance-optimized **personal portfolio website** showcasing professional experience, technical skills, academic background, and featured projects in **Data Science and Cyber Security**.

Designed with a strong focus on **clean UI/UX, accessibility, and scalability**, suitable for recruiters, hiring managers, and technical collaborators.

---

## Overview

This portfolio serves as a centralized professional profile highlighting:
- Academic background in **Cyber Security and Data Science**
- Industry experience in **IT Infrastructure & Network Support**
- Hands-on projects in **Machine Learning, Network Design, and Security**
- A modern, interactive, and responsive web interface

The project is built using **vanilla web technologies** without frameworks to ensure fast load times and maximum compatibility.

---

## Key Features

- Responsive design optimized for desktop, tablet, and mobile devices  
- Dark / Light theme toggle with persistent user preference  
- Animated scroll progress indicator  
- Smooth scrolling navigation  
- Typewriter animation for dynamic role highlights  
- Animate On Scroll (AOS) effects for enhanced visual flow  
- Interactive project showcase cards  
- Email-based contact form integration  
- Lightweight and framework-free implementation
